**NoteMaker** is a Gemini API-powered note maker built with Python, JavaScript, and React. Users input text, which is then processed by Google’s Gemini LLM to generate concise study notes. The app demonstrates AI-assisted note formatting, API integration, and interactive note management through a modern web interface.

## Description
<b>NoteMaker</b>
A Gemini API-powered note maker built with Python, JavaScript, and React. Users input text, which is then processed by Google’s Gemini LLM to generate concise study notes. The app demonstrates AI-assisted note formatting, API integration, and interactive note management through a modern web interface.
